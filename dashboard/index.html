Chào mừng tới Jarvis
Vài bước nhanh để bắt đầu. Mọi thứ đổi lại được sau trong ⚙ Cài đặt.

1. Workspace

2. Tài khoản admin (khuyến nghị)

Đặt mật khẩu để chặn người lạ — quan trọng nếu đưa lên VPS. Bỏ trống nếu chỉ chạy máy cá nhân.
❓ Mã thiết lập là gì? Jarvis đang chạy trên server công khai. Để chắc chính bạn (không phải người lạ vô tình mở link) là người tạo tài khoản admin, bạn dán một mã chỉ in ra log server — nơi duy nhất chủ máy xem được.
Cách lấy mã (Hostinger → App terminal):
  1. Mở app → bấm App terminal (bạn sẽ vào BÊN TRONG container jarvis).
  2. Chạy: cat /data/state/.setup_token
  3. Copy chuỗi hiện ra → dán vào ô trên.
Nếu SSH vào HOST (không phải App terminal): docker compose logs jarvis | grep "SETUP TOKEN"
💡 Khỏi cần mã: nếu lúc deploy bạn đặt sẵn env JARVIS_ADMIN_PASSWORD (+ JARVIS_ADMIN_USER), mở app ra là đăng nhập luôn, không hỏi mã.

3. Nhà cung cấp AI (bộ não)

Chọn 1 nhà cung cấp chính. Đổi lại được trong ⚙ Models.
Đăng nhập Jarvis
Quên mật khẩu?
Mở file server/settings.json, xóa khối "auth" (hoặc đặt rỗng), rồi chạy stop-jarvis.bat → start-jarvis.vbs. Mở lại sẽ về bộ cài đặt để tạo tài khoản mới.
